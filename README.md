A world-class, premium React-based portfolio showcasing my journey as a **JSDET @ ApraLabs**, Full-Stack Developer, and Automation Engineer. This platform features a high-end glassmorphism UI, interactive 3D particle backgrounds, and a mobile-first responsive design.

---

## ✨ Key Features

- **💎 Premium Aesthetics**: Modern Glassmorphism design system with custom HSL tailored color palettes.
- **✨ Dynamic Animations**: Smooth entrance animations (Fade-In-Up), hover micro-interactions, and a typing effect.
- **🎨 Visual DNA**: Integrated 3D particle background using Canvas and optimized performance.
- **📱 Ultra Responsive**: Fluid layout transitions across Mobile, Tablet, and Desktop.
- **🛠 Tech Stack Showcase**: Interactive skill bars and category-based technology grid.

---

## 🛠 Tech Stack

- **Frontend**: React.js, JavaScript (ES6+), HTML5, CSS3 (Vanilla Glassmorphism)
- **State Management**: React Hooks (useState, useEffect)
- **Routing**: React Router DOM v6
- **Animations**: CSS Keyframes & Framer Motion (Optional)
- **Deployment**: Netlify / Vercel

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/HimanshuGrahacharya/My_Portfolio_Using_Reactjs.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```
